# Progetti Web

Questa repository contiene diversi progetti web realizzati come esercizi di HTML, CSS e JavaScript. Di seguito sono riportate le directory e le loro descrizioni.

## Directory e Descrizioni

### 1. Prima Pagina (es_1-primaPagina)

- **Descrizione**: Una semplice pagina HTML con uno sfondo scuro, un titolo centrato e un'immagine di Jabba the Hutt da Star Wars.
- **Percorso**: `Web1/es_1-primaPagina/`

### 2. Pagina Personale (es_2-paginaPersonale)

- **Descrizione**: Un progetto più complesso che include:
  - Una pagina principale con un effetto di sfondo video.
  - Una pagina di "hacking" con effetti di testo che cambiano.
  - Una pagina di "sicurezza" con animazioni di testo.
  - Utilizzo di JavaScript per gestire transizioni e effetti.
- **Percorso**: `Web1/es_2-paginaPersonale/`

### 3. Liste e Tabelle (es_3)

- **Descrizione**: Esempi di diversi tipi di liste e tabelle HTML.
  - **Liste**: Include liste non ordinate, ordinate, annidate e con diversi tipi di numerazione.
  - **Tabelle**: Esempi di creazione e formattazione di tabelle HTML, comprese intestazioni, unione di celle, bordi e spaziatura.
- **Percorso**: `Web1/es_3/`

### 4. Form (es_4)

- **Descrizione**: Un esempio completo di un form HTML che include:
  - Campi di input per nome, cognome, email, età.
  - Selezione del linguaggio di programmazione.
  - Selezione del sesso.
  - Area di testo per descrizione.
  - Checkbox per accettazione della privacy policy.
  - Pulsanti di invio e reset.
  - Una pagina separata con la privacy policy completa.
- **Percorso**: `Web1/es_4/`
